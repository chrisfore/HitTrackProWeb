Getting Started

Create a team, add players with jersey numbers, then start tracking hits on the field.

Track Tab

  • Select a team and player from the dropdowns
  • Tap anywhere on the field to record a hit location
  • Choose the hit type (required), pitch type, and pitch location (optional)
  • When a player is selected, only that player's hits are shown
  • When no player is selected, all team hits are shown
  • Hit History on the right shows pitch breakdowns — tap an entry to highlight those hits on the field
  • Use "Undo Last Hit" to remove the most recent entry

Results Tab

  • View spray charts and statistics filtered by team, player, or date range
  • Use the search box to find players by number or name
  • Click a player name in "Hits by Player" to filter — click again to deselect
  • Export PDF generates a printable report of the current view

Settings Tab

  • Create, rename, or delete teams
  • Add or remove players (jersey number required, name optional)
  • Use the search box to find players quickly
  • Select "All Teams" to view players across all teams

Data Management

  • Export Player — export a single player's data
  • Export Team — export an entire team's data
  • Export All Data — export everything
  • Import Data — import a .hitdata or .json file
  • Exported .hitdata files are compatible with the Hit Track Pro iOS app

Data Storage

Your data is stored locally in your browser using IndexedDB. It persists across browser restarts and device reboots. However:

  • Clearing your browser data or site data will erase your data
  • Private/Incognito mode does not save data — it is wiped when the window closes
  • Different browsers maintain separate data (e.g. Chrome and Safari do not share data)
  • We recommend exporting your data regularly as a backup
